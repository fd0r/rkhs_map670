## Set up

This repository requires python, but also needs some libraries:

- pandas
- numpy
- scikit-learn
- matplotlib
- jupyter
- googledrivedownloader
- os

If you need to install some of them, you can simply execute ```pip install -r requirements.txt``` on your terminal.
